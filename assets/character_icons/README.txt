Character icons for the Characters tab.

Filenames use Mario Kart Wii character codes, for example:
  fk.png = Funky Kong
  ds.png = Daisy
  mr.png = Mario

These icons were generated from a user-provided sprite sheet for this local build. Do not upload copyrighted Nintendo artwork to a public GitHub repository unless you have the legal right to distribute it.

If you want a clean public release, delete the PNG files and keep this folder empty or use original artwork you own.
