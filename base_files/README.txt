Put your own Mario Kart Wii base files here.

Recommended: open the app, go to Setup, and click Extract base files from WBFS/ISO.
The app will copy only the required folders from your own legally dumped MKWii disc image.

Required layout:
base_files/Scene/UI
base_files/Scene/Model
base_files/Scene/Model/Kart
base_files/Race/Kart
base_files/Race/Course
base_files/Sound

Do not publish Nintendo base files with public releases.
Each user must provide their own MKWii dump or extracted files.
